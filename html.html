HTML is made using tags like <h1></h1>.
Simply, tags define sections and tell your computer to display/do.

then entering the heading's text and finally closing it off with the same tag but with a "/" in front of the

Or how <p></p> is used to create paragraphs on this page.

An important thing to remember is the standard structure of a html file and what each tag does.

This is the basic form of a html file (Indentation can be ignored).
The <!DOCTYPE html> tag can normally be ignored the most part.

<html> - Anything within these tags is the code of the html file.

<head> - Things within these tags are normally things that don't get displayed on the page itself and rather help build it.

<body> - Things within these tags are meant to be displayed on the page itself like this text!

Remember to close all your tags!

Key Tags

These are (hopefully) all the tags you need to know:

  • <p></p> - Used for making paragraphs.
  • <h1></h1> - Used for making headings. (Remember there is h2, h3, h4, h5 doing down in size from h1)
  • <div></div> - Used for grouping tags together.
  • <a></a> - Used for hyperlinks to other pages a website or external ones.
  • <ul></ul> - Used to create an unordered list (bullet points).
  • <ol></ol> - Used to create an ordered list (numbers).
  • <li></li> - Used to add a entry to any list.
  • <link/> - Used to attach a stylesheet to the html file.
  • <img> - Used to display a image from a source.
  • <video> - Used to display a video from a source.
  • <audio> - Used to display a audio from a source.

Special Tags Explained

Some tags like <video> and <link/> have to setup in a way that is different than other tags.

For a start lets look at <a></a>

Anything within then will be displayed on the page and can be clicked to travel to the url set.
To set the link, enter href="https://www.example.com".
After doing so you can click on it.

<img>

Images have 3 common attributes, src, width, height.
src is used showing the path to the image file. The image doesn't have to be in the same folder as the html file.
Width and height are self explanatory.

<video></video>

Videos have the same 3 as before with a new controls one.
Including this allows the user to control the video.
autoplay can be used to auto play the video when the user enters the page.

<audio></audio>

Audio is pretty much the same as video expect its audio.

<link/>

Finally link is only used for linking the stylesheet to a html file.
It has 3 attributes, href, type, rel.
href is the source of the file (location).
type should be put as "text/css" (can be ignored)
rel should be put as "stylesheet" (can be ignored?)
